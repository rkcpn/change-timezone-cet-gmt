# Change timezone CET <-> GMT
A simple bash script to quickle change timezone to either CET or GMT on Linux.

## Install
1. clone this repo
2. make chtmzn.sh executable: 'chmod +x chtmzn.sh'

## Usage
'./chtmzn.sh [-c cet] [-g gmt] [-h help]'

Options:
    -c, --cet:		change to CET
    -g, --gmt:		change to GMT
    -h, --help:		get this message
